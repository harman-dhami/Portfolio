Unity Game
By using Unity's 3D game engine, I created a FPS game. The game incorporates custom C# scripts,
which control the many movements in the game. The game also features a main menu, options menu,
end screen. Many components are customizable due to the C# scripts.

Java Projects
I have created multiple projects using Java. Some examples include a triangle test program, a distance
calculator, and a program which puts a ball into a bean machine. The triangle test program accepts inputs
for the sides, and returns the properties of the triangle. The distance calculator takes in the x and y
coordinates of a point and calculates the distance between it and another point. The bean machine program
asks the user for number of balls and returns the path they took back to the user.
